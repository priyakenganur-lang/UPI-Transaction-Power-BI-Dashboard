# UPI-Transaction-Poer-BI-Dashboard
Project Overview

 This project focuses on analyzing UPI (Unified Payments Interface) transaction data using Microsoft Power BI. The dashboard provides a comprehensive overview of transaction patterns, including payment methods, bank performance, customer demographics, and transaction status. It helps users gain meaningful insights into digital payment trends and supports data-driven decision-making.
________________________________________
Objectives

•	 To analyze UPI transaction trends across different cities and banks

•	To identify success and failure rates of transactions

•	To understand customer behavior based on gender and device usage

•	To evaluate popular payment methods and transaction purposes

•	To create an interactive dashboard for easy data exploration

Dataset Description

The The dataset contains UPI transaction records with the following key fields:

•	Transaction ID

•	Transaction Date & Time

•	Sender and Beneficiary Bank Names

•	City (e.g., Bangalore, Delhi, Hyderabad)

•	Transaction Type (Payment / Transfer)

•	Payment Method (UPI ID, Phone Number, QR Code)

•	Amount and Currency

•	Transaction Status (Success / Failed)

•	Customer Demographics (Age, Gender)

•	Device Type (Mobile, Tablet, Laptop)

•	Purpose (Shopping, Travel, Bills, Food, etc.)
________________________________________
Data Pipeline

1.	 Data Collection – Gathered transaction dataset

2.	Data Cleaning – Removed duplicates, handled missing values

3.	Data Transformation – Structured data using Excel/Power BI

4.	Data Modeling – Created relationships between fields

5.	Visualization – Designed interactive dashboard in Power BI

Dashboard Features

 •	KPI Cards:

  o	Total Transactions

  o	 Total Amount

 •	Visualizations:

  o	Donut charts (Gender, Payment Status)

  o	Bar charts (Payment Method, City-wise Amount)

  o	Tree maps (Transaction Purpose & Merchants)

  o	Drill-down and filtering options for deeper insights

•	Filters (Slicers):

  o	Transaction Date (Date range slider)

  o	Sender Bank Name (Axis Bank, HDFC, ICICI, SBI)

  o	Beneficiary Bank Name (Axis Bank, HDFC, ICICI, SBI)

  o	City (Bangalore, Delhi, Hyderabad)

  o	Transaction Type (Payment, Transfer)
________________________________________
Key Insights

•	 Equal participation observed between male and female users

•	Majority of transactions are successful (~80%)

•	UPI ID and QR Code are the most used payment methods

•	Major transaction volumes come from metro cities like Bangalore and Delhi

•	Shopping, Travel, and Bill Payments are top transaction categories

•	Mobile devices dominate transaction usage
